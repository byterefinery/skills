# C4 Diagrams

C4 diagrams model software architecture at multiple levels of abstraction: Context, Container, and Component.

## Declaration

Use `C4Context`, `C4Container`, `C4Component`, `C4Dynamic`, or `C4Deployment`.

```mermaid
    C4Context
      Person(user, "User")
      System(app, "Application")
      Rel(user, app, "Uses")
```

## Context Diagram

Show the system and its users/stakeholders.

```mermaid
    C4Context
      title System Context Diagram
      Enterprise_Boundary(bank, "Banking") {
        Person(customer, "Customer", "A bank customer")
        System(bankingApp, "Banking App", "Online banking system")
        SystemDb_Ext(mainframe, "Mainframe", "Core banking system")
      }
      Rel(customer, bankingApp, "Uses")
      Rel(bankingApp, mainframe, "Queries")
```

## Container Diagram

Show the high-level technology containers (web app, API, database).

```mermaid
    C4Container
      title Container Diagram
      Person(user, "User")
      System_Boundary(web, "Web Application") {
        Container(spa, "Single Page App", "JavaScript/React", "Runs in browser")
        Container(api, "API", "Java/Spring Boot", "Provides REST API")
        ContainerDb(db, "Database", "PostgreSQL", "Stores application data")
      }
      Rel(user, spa, "Uses")
      Rel(spa, api, "Calls")
      Rel(api, db, "Reads/Writes")
```

## Component Diagram

Show components within a container.

```mermaid
    C4Component
      title Component Diagram
      System(sys, "Banking System")
      Container_Boundary(webApp, "Web Application") {
        Component(controller, "REST Controller", "Handles HTTP requests")
        Component(service, "Business Service", "Contains business logic")
        Component(repo, "Repository", "Data access layer")
      }
      Rel(controller, service, "Uses")
      Rel(service, repo, "Uses")
```

## Styling and Layout

Use `UpdateElementStyle` and `UpdateRelStyle`. Set layout with `UpdateLayoutConfig`.

```mermaid
    C4Context
      title Styled Context
      Person(admin, "Admin")
      System(app, "Application")
      Rel(admin, app, "Manages")
      UpdateElementStyle(admin, $fontColor="red", $bgColor="grey")
      UpdateRelStyle(admin, app, $textColor="blue", $lineColor="blue")
      UpdateLayoutConfig($c4ShapeInRow="2", $c4BoundaryInRow="1")
```
