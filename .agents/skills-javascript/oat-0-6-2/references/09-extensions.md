# Extensions

Community third-party extensions that work with Oat.

## oat-chips

Chip/tag component with dismissible filters, colors, and toggle selection.

- **Repository**: https://github.com/someshkar/oat-chips
- **Demo**: https://oat-chips.somesh.dev
- **Size**: ~1KB gzipped

## oat-animate

Lightweight animation extension with declarative `ot-animate` triggers.

- **Repository**: https://github.com/dharmeshgurnani/oat-animate
- **Demo**: https://oat-animate.dharmeshgurnani.com
- **Size**: ~1KB gzipped
- **Features**: `on-load`, `hover`, `in-view` triggers, reduced-motion support

## oat-table

Semantic table enhancement — sort, filter, and select rows without framework widgets.

- **Repository**: https://github.com/MADEVAL/Oat-Table
- **Demo**: https://demo.globus.studio/oat-table/

## oat-upload

Dropzone, file previews, validation, removal, and progress for native `<input type="file">`.

- **Repository**: https://github.com/MADEVAL/Oat-Upload
- **Demo**: https://demo.globus.studio/oat-upload/
