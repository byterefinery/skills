# Writing and Exporting

## Core Write Functions

### `XLSX.write(workbook, opts)`

Returns file data in the specified format. Requires `bookType` and `type` options:

```js
const buffer = XLSX.write(wb, { type: "buffer", bookType: "xlsx" });
const base64 = XLSX.write(wb, { type: "base64", bookType: "xlsx" });
const binary = XLSX.write(wb, { type: "binary", bookType: "xlsx" });
const array = XLSX.write(wb, { type: "array", bookType: "xlsx" });
const str = XLSX.write(wb, { type: "string", bookType: "csv" }); // text formats only
```

### `XLSX.writeFile(workbook, filename, opts?)`

Writes directly to a file. `bookType` inferred from extension if not specified:

```js
XLSX.writeFile(wb, "output.xlsx");     // xlsx
XLSX.writeFile(wb, "output.csv");      // csv
XLSX.writeFile(wb, "output.ods");      // ods
XLSX.writeFile(wb, "output.xlsb");     // xlsb
XLSX.writeFile(wb, "output.html");     // html
```

### `XLSX.writeXLSX` / `XLSX.writeFileXLSX`

Limited versions that only support XLSX format. Smaller bundle for sites that only export XLSX:

```js
const buffer = XLSX.writeXLSX(wb, { type: "buffer" });
XLSX.writeFileXLSX(wb, "output.xlsx");
```

## Output Type Encodings

| `type` | Output |
|---|---|
| `base64` | Base64-encoded string |
| `binary` | Binary string (byte `n` = `data.charCodeAt(n)`) |
| `string` | JS string (text formats only: CSV, HTML, etc.) |
| `buffer` | Node.js Buffer or Uint8Array |
| `array` | ArrayBuffer or array of uint8 |
| `file` | Attempt to download a file |

## Compression

XLSX, XLSB, NUMBERS, and ODS use ZIP containers. By default, SheetJS optimizes for speed (larger files). Enable compression for smaller files:

```js
XLSX.writeFile(wb, "output.xlsx", { compression: true });
```

## Shared String Table (SST)

By default, text cells use "inline" strings. For better compatibility with iOS Numbers and third-party tools, use SST:

```js
XLSX.writeFile(wb, "output.xlsx", { bookSST: true });
```

SST is slower and more memory-intensive but produces smaller files with better compatibility.

## Workbook Properties Override

```js
XLSX.writeFile(wb, "output.xlsx", {
  Props: {
    Title: "Monthly Report",
    Subject: "Sales",
    Author: "SheetJS",
    Company: "SheetJS Inc",
    Keywords: "report, sales",
    Comments: "Generated by SheetJS",
  }
});
```

## Browser Download Patterns

### Blob Download

```js
const buffer = XLSX.write(wb, { type: "array", bookType: "xlsx" });
const blob = new Blob([buffer], { type: "application/octet-stream" });
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "output.xlsx";
a.click();
URL.revokeObjectURL(url);
```

### Using `writeFile` in Browser

`XLSX.writeFile` works in browsers — it triggers a download:

```js
XLSX.writeFile(wb, "output.xlsx");
```

## NUMBERS Export

Requires the `xlsx.zahl` payload:

```js
// Node.js
const XLSX = require("xlsx");
const XLSX_ZAHL_PAYLOAD = require("xlsx/dist/xlsx.zahl");

// ESM
import * as XLSX from "xlsx";
import XLSX_ZAHL_PAYLOAD from "xlsx/dist/xlsx.zahl.mjs";

// CDN
// <script src="https://cdn.sheetjs.com/xlsx-0.20.3/package/dist/xlsx.zahl.js"></script>
// XLSX_ZAHL_PAYLOAD is available as global

XLSX.writeFile(wb, "output.numbers", {
  numbers: XLSX_ZAHL_PAYLOAD,
  compression: true
});
```

## CSV Output Notes

- `XLSX.write` with `bookType: "csv"` always includes UTF-8 BOM for Excel compatibility
- `XLSX.utils.sheet_to_csv()` returns a plain string without BOM
- Use `FS` option for custom field separator (e.g., `FS: "\t"` for TSV)
- Use `RS` option for custom record separator

## VBA Export

```js
// Re-attach VBA from parsed file
const wb = XLSX.read(data, { bookVBA: true });
XLSX.writeFile(wb, "output.xlsm", { bookVBA: wb.vbaraw });
```
