# requests 2.34.2 — API Reference

## Module-Level Functions

All functions in `requests.api` delegate to a temporary `Session`. They share the same parameter set via `request()`:

### `request(method, url, **kwargs)`

The universal entry point. All other module-level functions call this internally.

| Parameter | Type | Default | Description |
|---|---|---|---|
| `method` | `str` | — | HTTP method: `GET`, `POST`, `PUT`, `PATCH`, `DELETE`, `HEAD`, `OPTIONS` |
| `url` | `str \| bytes` | — | Target URL |
| `params` | `dict \| list[tuple] \| bytes \| str` | `None` | Query string parameters (URL-encoded) |
| `data` | `dict \| list[tuple] \| bytes \| file-like` | `None` | Request body (form-encoded or raw) |
| `json` | `JSON-serializable` | `None` | JSON body (auto-serialized, sets `Content-Type: application/json`) |
| `headers` | `dict` | `None` | HTTP headers |
| `cookies` | `dict \| CookieJar` | `None` | Cookies to send |
| `files` | `dict[str, tuple]` | `None` | Multipart file uploads |
| `auth` | `tuple \| AuthBase` | `None` | Authentication handler |
| `timeout` | `float \| tuple[float, float]` | `None` | Connect and/or read timeout in seconds |
| `allow_redirects` | `bool` | `True` | Follow redirects (False for HEAD by default) |
| `proxies` | `dict[str, str]` | `None` | Protocol→proxy URL mapping |
| `verify` | `bool \| str` | `True` | TLS verification (path to CA bundle or boolean) |
| `stream` | `bool` | `False` | Defer downloading response body |
| `cert` | `str \| tuple[str, str]` | `None` | Client certificate path or (cert, key) tuple |
| `hooks` | `dict[str, callable]` | `None` | Event hooks (`{"response": fn}`) |

**Returns:** `Response`

### Convenience Functions

- **`get(url, params=None, **kwargs)`** — GET request
- **`post(url, data=None, json=None, **kwargs)`** — POST request
- **`put(url, data=None, **kwargs)`** — PUT request
- **`patch(url, data=None, **kwargs)`** — PATCH request
- **`delete(url, **kwargs)`** — DELETE request
- **`head(url, **kwargs)`** — HEAD request (`allow_redirects=False` by default)
- **`options(url, **kwargs)`** — OPTIONS request

## Core Classes

### `Request(method, url, headers, files, data, params, auth, cookies, hooks, json)`

User-facing request builder. Call `.prepare()` to get a `PreparedRequest`.

```python
req = Request("GET", "https://example.com", params={"q": "test"})
prepared = req.prepare()
# Or via session:
prepared = session.prepare_request(req)  # merges session defaults
```

### `PreparedRequest`

The fully-serialized request ready for transmission. Generated by `Request.prepare()` or `Session.prepare_request()`.

Attributes:
- `method` — HTTP method string
- `url` — Full URL with query string
- `headers` — `CaseInsensitiveDict` of headers
- `body` — Request body as bytes (or file-like)
- `hooks` — Hook callbacks dict

```python
prep = PreparedRequest()
prep.prepare(method="POST", url=url, json={"key": "val"})
session.send(prep, timeout=5.0)
```

### `Response`

Server response object.

| Attribute | Type | Description |
|---|---|---|
| `status_code` | `int` | HTTP status code (e.g. 200, 404) |
| `headers` | `CaseInsensitiveDict` | Response headers |
| `content` | `bytes` | Raw response body bytes |
| `text` | `str` | Decoded response body string |
| `encoding` | `str \| None` | Explicit encoding; set before accessing `.text` |
| `apparent_encoding` | `str \| None` | Detected encoding via charset-normalizer/chardet |
| `cookies` | `RequestsCookieJar` | Server-set cookies |
| `elapsed` | `timedelta` | Time from request sent to response received |
| `history` | `list[Response]` | Redirect chain (oldest first) |
| `request` | `PreparedRequest` | The request that produced this response |
| `reason` | `str \| None` | HTTP reason phrase ("OK", "Not Found") |
| `url` | `str` | Final URL (after redirects) |
| `raw` | `urllib3.response.HTTPResponse` | Underlying urllib3 response (with `stream=True`) |
| `ok` | `bool` | True if status < 400 |
| `is_redirect` | `bool` | True if response is a redirect with Location header |
| `is_permanent_redirect` | `bool` | True for 301/308 redirects |
| `links` | `dict` | Parsed Link header (pagination, etc.) |
| `next` | `PreparedRequest \| None` | Next request in redirect chain (when `allow_redirects=False`) |

Methods:
- `.json(**kwargs)` — Parse body as JSON; raises `JSONDecodeError` on failure
- `.raise_for_status()` — Raise `HTTPError` on 4xx/5xx status
- `.iter_content(chunk_size, decode_unicode)` — Iterator over bytes chunks
- `.iter_lines(chunk_size, decode_unicode, delimiter)` — Iterator over lines
- `.close()` — Release connection back to pool
- Context manager: supports `with` statement

### `CaseInsensitiveDict`

Dict subclass with case-insensitive key lookup. Used for request/response headers.

```python
from requests.structures import CaseInsensitiveDict
cid = CaseInsensitiveDict({"Content-Type": "text/html"})
cid["content-type"] == "text/html"  # True
cid["CONTENT-TYPE"] == "text/html"  # True
list(cid) == ["Content-Type"]        # preserves original case
```

## Status Codes

Accessed via `requests.codes`:

```python
requests.codes.ok                    # 200
requests.codes.created               # 201
requests.codes.no_content            # 204
requests.codes.moved_permanently     # 301
requests.codes.found                 # 302
requests.codes.not_modified          # 304
requests.codes.temporary_redirect    # 307
requests.codes.bad_request           # 400
requests.codes.unauthorized          # 401
requests.codes.forbidden             # 403
requests.codes.not_found             # 404
requests.codes.teapot                # 418
requests.codes.internal_server_error # 500
requests.codes.bad_gateway           # 502
requests.codes.service_unavailable   # 503

# Multiple aliases exist: codes.ok == codes.okay == codes.all_ok
# Access by dict key: requests.codes['temporary_redirect']
```

## Constants

| Constant | Value | Description |
|---|---|---|
| `DEFAULT_REDIRECT_LIMIT` | 30 | Max redirects before `TooManyRedirects` |
| `REDIRECT_STATI` | `(301, 302, 303, 307, 308)` | Auto-processable redirect codes |
| `CONTENT_CHUNK_SIZE` | 10240 | Default chunk size for `.content` |
| `ITER_CHUNK_SIZE` | 512 | Default chunk size for `.iter_lines()` |

## `requests.packages`

Namespace for accessing requests' dependencies (urllib3, charset-normalizer/chardet, idna). Useful when you need urllib3 types without a separate import:

```python
from requests.packages import urllib3
from requests.packages.urllib3.util.retry import Retry
```

Note: Direct imports (`import urllib3`) are preferred; `requests.packages` is provided for compatibility.
