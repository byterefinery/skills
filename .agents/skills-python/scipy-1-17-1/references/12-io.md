## MATLAB Files (.mat)

### `loadmat(filename, ...)`

Read a MATLAB .mat file. Returns an `OrderedDict` mapping variable names to NumPy arrays.

```python
from scipy.io import loadmat

data = loadmat('file.mat')
print(data.keys())  # variable names
array = data['variable_name']
```

**Key parameters:**
- `appendmat=True` — include MATLAB header info in keys
- `variable_names=None` — read only specified variables
- `struct_as_record=True` — return structs as record arrays
- `mat_version=None` — auto-detect (supports v4, v7, v7.3)

### `savemat(filename, mdict, ...)`

Write a dictionary of variables to a .mat file.

```python
from scipy.io import savemat
import numpy as np

data = {
    'x': np.linspace(0, 1, 100),
    'y': np.sin(2 * np.pi * 5 * np.linspace(0, 1, 100)),
    'metadata': {'author': 'me', 'date': '2024-01-01'}
}
savemat('output.mat', data)
```

**Key parameters:**
- `format='v7.3'` — HDF5-based format (supports large files, v7.3 only)
- `format='v7'` — default, standard MATLAB format
- `do_compression=False` — disable compression
- `oned_as='row'` — save 1-D arrays as row vectors (`'column'` for column vectors)

### `whosmat(filename)`

List variables in a .mat file without loading data. Returns list of `(name, shape, type)` tuples.

```python
from scipy.io import whosmat

info = whosmat('file.mat')
for name, shape, dtype in info:
    print(f"{name}: shape={shape}, dtype={dtype}")
```

## Matrix Market Files

Matrix Market is a standard format for sparse and dense matrices.

### `mmread(filename)`

Read a Matrix Market file. Returns a dense NumPy array or sparse matrix.

```python
from scipy.io import mmread

matrix = mmread('matrix.mtx')
```

### `mmwrite(filename, A, field=None, comment=None)`

Write a matrix to Matrix Market format.

```python
from scipy.io import mmwrite

mmwrite('output.mtx', sparse_matrix, field='real', comment='My matrix')
```

**Field options:** `'real'`, `'complex'`, `'pattern'`, `'integer'`, `'double'`.

### `mminfo(filename)`

Query matrix info without loading. Returns `(is_complex, is_symmetric, is_pattern, is_square)`.

```python
from scipy.io import mminfo

is_complex, is_symmetric, is_pattern, is_square = mminfo('matrix.mtx')
```

## NetCDF Files

NetCDF (Network Common Data Form) for scientific data.

### `netcdf_file(filename, mode='r', ...)`

File object for NetCDF data (classic format).

```python
from scipy.io import netcdf_file
import numpy as np

# Read
with netcdf_file('data.nc', 'r') as nc:
    print(nc.variables.keys())  # variable names
    data = nc.variables['temperature'][:]  # read all data
    lat = nc.variables['lat'][:]
    lon = nc.variables['lon'][:]

# Write
with netcdf_file('output.nc', 'w', format='CLASSIC') as nc:
    nc.createDimension('x', 100)
    nc.createDimension('y', 50)
    var = nc.createVariable('data', 'f4', ('x', 'y'))
    var[:] = np.random.rand(100, 50)
    var.long_name = 'My Data'
```

### `netcdf_variable`

Data object representing a NetCDF variable (returned by `netcdf_file`).

**Note:** For newer NetCDF files (64-bit offset, 64-bit data), consider using the `netCDF4` Python package instead.

## Unformatted Fortran Files

### `FortranFile(file, dtype='<f8', ...)`

Read/write unformatted sequential Fortran files (with record markers).

```python
from scipy.io import FortranFile
import numpy as np

# Write
with FortranFile('output.bin', 'w') as ff:
    ff.write_arrays([np.array([1, 2, 3]), np.array([[1, 2], [3, 4]])])

# Read
with FortranFile('input.bin', 'r') as ff:
    arrays = ff.read_vars()
    # or read specific dtypes:
    data = ff.read_floats(np.float64)
```

**Methods:**
- `write_arrays(arrays)` — write multiple arrays
- `read_vars()` — read all variables
- `write_floats(array)` / `read_floats(dtype)` — float arrays
- `write_ints(array)` / `read_ints(dtype)` — integer arrays
- `write_complexes(array)` / `read_complexes(dtype)` — complex arrays
- `write_records(*arrays)` — write with record markers
- `read_records(dtype)` — read records

## WAV Sound Files

### `scipy.io.wavfile`

| Function | Description |
|---|---|
| `wavfile.read(filename)` | Read WAV file → `(rate, data)` |
| `wavfile.write(filename, rate, data)` | Write WAV file |

```python
from scipy.io import wavfile

# Read
sample_rate, audio_data = wavfile.read('audio.wav')
print(f"Sample rate: {sample_rate} Hz, Duration: {len(audio_data)/sample_rate:.2f}s")

# Write
wavfile.write('output.wav', 44100, audio_data)
```

**Notes:**
- Supports 8-bit unsigned, 16-bit signed, 24-bit signed, and 32-bit signed/float
- Returns data as int16 (for 16-bit), int32 (for 24/32-bit), or float32
- Mono and stereo supported

## Harwell-Boeing Files

| Function | Description |
|---|---|
| `hb_read(filename)` | Read Harwell-Boeing file → `(data, info_dict)` |
| `hb_write(matrix, filename, ...)` | Write Harwell-Boeing file |

```python
from scipy.io import hb_read

matrix, info = hb_read('matrix.hb')
print(info['problem'])  # problem description
```

## ARFF Files (Weka)

ARFF (Attribute Relation File Format) used by Weka ML tool.

| Function | Description |
|---|---|
| `arff.loadarff(filename)` | Load ARFF file → `(data, meta_info)` |
| `arff.topandas(data, meta_info)` | Convert to pandas DataFrame |

```python
from scipy.io import arff

data, meta = arff.loadarff('dataset.arff')
# data is a structured NumPy array
# meta describes attribute types
```

## IDL Save Files

### `readsav(filename)`

Read an IDL (Interactive Data Language) .sav file. Returns a dictionary-like object.

```python
from scipy.io import readsav

data = readsav('file.sav')
print(data.variable_name)
```
